# Godot Action RPG Prototype

This project is a small action RPG prototype built with Godot 4.  
It was developed as a learning project focused on implementing core RPG systems and scene structure.

## Features

- Basic stat system (STR, DEX, VIT)
- PlayerStats module for stat calculations
- Player movement and dash skill
- Enemy entities with basic combat interaction
- Dungeon and village scenes
- HUD and status UI
- GameManager for global game state handling

## Project Structure

PlayerData → persistent player data  
PlayerStats → stat calculations  
Player → gameplay logic and controls  
GameManager → global state and scene transitions  

## Engine

Godot 4.x

## Status

Prototype / learning project
